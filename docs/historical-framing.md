# 歴史表現方針

## 基本方針

本コンテンツは、道後公園・湯築城跡を題材にした「史実に着想を得た体験型復元」です。厳密な考古学的復元や建築史的な再現ではなく、来訪者が戦国期の城郭、武家屋敷、侍や足軽の暮らしを想像しやすくするための教育・観光向けプロトタイプとして扱います。

## 湯築城跡を題材にする理由

湯築城跡は、近世の巨大天守を中心に見せる城郭とは異なり、土塁、堀、居住空間、地域支配の拠点としての性格を伝えやすい題材です。360度ウォークスルーでは、城を「眺める対象」ではなく「歩いて理解する空間」として提示できます。

## 厳密復元ではなく体験型復元であること

画面とREADMEでは、以下の趣旨を明示します。

```text
本コンテンツは、道後公園・湯築城跡を題材とした歴史的イメージ再現です。
厳密な史実復元・考古学的復元を示すものではなく、教育・観光・展示用途を想定した体験型プロトタイプです。
```

## 避けるべき表現

- 近世城郭風の巨大天守
- 姫路城・大阪城のような大規模白亜天守
- 現代の道路、アスファルト、電柱、看板、自動車
- 現代服
- ファンタジー要素
- 西洋風の城
- 過度に豪華な御殿
- 不自然な漢字看板・英語看板
- 史実と無関係な魔法的表現

## 展示利用時の説明

展示やイベントで使う場合は、画像生成AIによる仮想再現であること、湯築城跡の現地理解を補助する体験であること、現地の資料や専門的な解説と併用することを案内します。
