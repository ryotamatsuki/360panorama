```text
N01 外堀入口
  ↓
N02 大手門前
  ↓
N03 土塁沿いの通路
  ↓
N04 城内広場
  ↓

  ↓

  ↓
N07 詰所・見張り場
  ↓
```

## 各地点の役割

| 地点 | 役割 | 表示の中心 |

| N01 外堀入口 | 導入 | 堀、土塁、木橋、遠くの門 |
| N02 大手門前 | 境界と緊張感 | 木造門、門番、足軽 |
| N03 土塁沿いの通路 | 城郭空間の連続性 | 土塁、木柵、堀、巡回 |
| N04 城内広場 | 生活感と活動感 | 井戸、馬、物資、家人 |
| N05 武家屋敷前の通り | 居住空間 | 屋敷門、低い塀、庭木 |
| N06 武家屋敷の庭と縁側 | 内部体験 | 縁側、庭石、槍立て、炊事 |
| N07 詰所・見張り場 | 軍事的緊張感 | 見張り台、旗、槍、弓 |
| N08 高台から見た城内の眺望 | 締め | 土塁、堀、門、屋敷群の俯瞰 |

## 隣接地点との共有ランドマーク

- N01/N02: 外堀、木橋、大手門を共有する。
- N02/N03: 門の内側、木柵、土塁への進路を共有する。
- N03/N04: 土塁の終端、城内へ開ける通路を共有する。
- N04/N05: 広場脇の屋敷群、物資運搬の動線を共有する。
- N05/N06: 屋敷門、低い塀、庭木を共有する。
- N06/N07: 屋敷裏手の通路、武具の気配を共有する。
- N07/N08: 見張り場から高台へ上がる視点、旗や柵を共有する。

## yaw / pitch 調整メモ

現在の `src/tour-data.js` に入っている `initialYaw`、`initialPitch`、各リンクの `yaw`、`pitch` はプレースホルダー画像用の仮値です。本番の2:1 equirectangular panoramaへ差し替えた後は、実際の構図に合わせて必ず再調整してください。

- `initialYaw`: シーンを開いた直後に見せたい正面方向。
- `initialPitch`: シーンを開いた直後の上下方向。通常は0付近でよい。
- リンクの `yaw`: ホットスポットを置く左右方向。次地点へ続く道や門の位置に合わせる。
- リンクの `pitch`: ホットスポットを置く上下方向。地面付近の進路ならやや負の値にする。

調整時は、隣接地点の共有ランドマークが「背後に前地点、前方に次地点」として自然につながるかを確認します。
